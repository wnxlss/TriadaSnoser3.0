# TriadaSnoser 3.0

TRIADA — Telegram Account Terminator TRIADA — это высокопроизводительный инструмент для автоматического массового удаления («сноса») аккаунтов из чатов и каналов Telegram. Разработан для модераторов, администраторов и владельцев сообществ, нуждающихся в глубокой очистке аудитории.

Ключевые возможности:

Массовое удаление: Быстрое исключение большого количества участников по заданным критериям.

Гибкая фильтрация: Выбор целей по дате регистрации, последней активности, отсутствию username, статусу «бота» или «скама».

Анонимность и безопасность: Работа через ваши собственные сессии (user accounts). Не требует API ID/Hash бота, использует MTProto.

Высокая производительность: Полностью асинхронная реализация на Python (aiogram, Telethon) для максимальной скорости.

Обход ограничений: Интеллектуальные задержки и обработка ошибок для минимизации блокировок со стороны Telegram.

Логирование: Подробные логи о процессе для анализа и отладки.

Для кого это: Админы крупных каналов, модераторы борьбы со спамом, исследователи социальной инженерии (в образовательных целях).

⚠️ Важно: Используйте инструмент ответственно. Соблюдайте правила Telegram и законодательство вашей страны. Разработчик не несет ответственности за нарушение ToS.
